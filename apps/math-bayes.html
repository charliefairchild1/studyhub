PEP connection. The posterior density IS the model weight. Each observed flip is a residual (the event that happened, against what the current posterior would have predicted). Bayesian update concentrates probability mass on values that survive the data — which is exactly what gradient descent does with continuous weights. See 📈 Predictor for the gradient version; this page shows the probability-measure version.
