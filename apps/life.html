The math

Each rule is a function from a cell's current state + its 8 neighbors → next state. Tiny rule. But apply it simultaneously to every cell, every generation, and you get gliders (stable moving patterns), guns (emitters), oscillators, and complete chaos. Conway's Life is Turing-complete — you can build a universal computer from cells following B3/S23.

The math question: given a rule, which initial states produce stable order, which produce chaos, and which produce complexity (order + chaos together)? Wolfram's classification.

PEP primitives used

Weighted graph — each cell is a node; the 8-neighborhood defines the edges (uniform weight 1).
State modulation — each cell's state at t+1 depends on its state at t modulated by its neighborhood's collective state. That's the primitive applied to a tessellated substrate.
Opacity / haze — in Conway's Life, a live cell with fewer than 2 or more than 3 neighbors dies. That's an opacity rule triggered by local conditions: the cell's "encoding strength" drops below reuse threshold. Brian's Brain makes this literal — the 3-state "dying" phase IS decaying opacity.

Why this matters for PEP: complex behavior emerging from simple local rules on a weighted graph is the whole thesis. If PEP's primitives generate this much structure on a 2D tessellation with boolean states, the same primitives on continuous-valued embedding graphs produce real cognition.
