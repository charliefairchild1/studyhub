Temperature is state modulation. T rescales the accept function — the same objective, the same moves, but the optimizer's tolerance for dissipation changes with T. High T lets the system cross energy barriers (escape local minima); low T locks it into whatever basin it's in. This is Axona's mood/arousal dial in disguise: same options, different acceptance thresholds.

PTO reading: tolerating short-term dissipation (T > 0) is a precondition for long-term transformation. A greedy policy optimizes local PTO and gets stuck; annealed policy pays dissipation cost to find the better basin.
