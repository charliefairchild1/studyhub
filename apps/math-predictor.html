What this is

A signal streams in over time. Two predictors guess the next value: naive (persistence — predicts last-seen) and adaptive (linear model with online gradient descent). The residual — observed minus predicted — feeds back to update the adaptive model's weights. Watch its error shrink while the naive predictor's stays flat. This IS PEP's learning loop.

PEP primitives used

Predictor + residual — the headline. The adaptive linear model IS a predictor; the error between prediction and observation IS the residual; the weight update IS learning.
Weighted graph — the window of past values is a mini-graph where each past time-step is a node weighted by how much it predicts the next one. Those weights are what get learned.

Why this matters: every adaptive component in PEP (Vectora's retrieval relevance, Atria's matching, Axona's state-mapping, Lingora's translate-bank) uses this exact loop. Residuals are the only training signal PEP ever needs.
