Opacity / Haze

Memory slots with encoding strength. Strength decays each tick; clicking rehearses (bumps strength). New memories stream in — when the grid fills, the weakest slot gets overwritten. Forgetting is the feature.

PEP primitive 5. Every node has an encoding strength that decays. Reuse (rehearsal) keeps it alive. Capacity is finite — forgetting must happen, and it happens first to what hasn't proven itself useful. This is how Lingora's spaced-repetition schedules constellations, how Axona marks memories consolidated during sleep, and how Vectora makes room in a bounded index.

Try this

Cram: high input, can't rehearse fast enough. Recent memories overwrite old ones before they consolidate.
Sleep: input stops, recently-rehearsed memories get a boost (consolidation).
Click things repeatedly: they glow bright and resist forgetting.
Lower threshold: fewer overwrites, faster capacity saturation.
