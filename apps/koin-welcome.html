The Ledger
of Minds

A manifesto for Koin. The complete reading list — main document, distilled versions, companions, translations, and notes for builders, funders, educators, and skeptics.

All documents are in ~/projects/koin/. Click any title below to open it. Each document has a ⎙ Print / PDF button in the top-right for printing.

The core texts
The Ledger of Minds — The full manifesto
v2.3 · ~145 pages · ~57,000 words

Eleven chapters and four appendices. Full Lagrangian formulation. Math with plain-language unpacking. Technical reference, stories from the imagined future, letters, deployment scenarios, FAQ. The complete argument.
The Essential Version
v1.0 · ~30 pages

The same argument distilled. For sharing with someone who won't read the long version. Eleven sections covering diagnosis, shift, what Koin is, the math, trust, theft, the world it builds, stories, lineage, objections, and the ask.
The Pamphlet
v1.0 · ~5 pages

The shortest serious version. For mass distribution, link-sharing, flyers, the table at the event. Five pages that carry the whole argument in compressed form.
El Panfleto (en español)
v1.0 · ~5 páginas

El mismo panfleto, traducido. Primer paso hacia el alcance multilingüe que el manifiesto eventualmente necesitará.
Koin for Kids
v1.0 · ~12 pages

Written for ages ten to fourteen. Story-driven, accessible vocabulary, empowering tone. Useful for teaching the concept to younger people — and for anyone who wants the friendliest version.
The companions
The Critics
v1.0 · ~30 pages

Eight serious critiques of Koin from eight intellectual traditions — Marxist, libertarian, feminist, conservative, AI safety, environmentalist, postcolonial, macroeconomist — each with a substantive response. The test the argument has to pass.
The Builder's Guide
v1.0 · ~25 pages

Concrete protocol specification for implementing a Koin pilot. did:koin identity, graph schema, attribution engine, settlement adapters, federation API, audit interface. Engineer-readable, with plain-language explanations alongside.
New Ideas — Proposed additions
v1.0 · ~25 pages

Ten proposed additions for future revisions of the manifesto. The Koin Will, the Lonely Voice, Memory Rights, the Care Parallel, Ecological Koin, the Conservative Case, Koin and Disability, Koin in Crisis, the Honoring, the First Question.
Teaching Koin — A guide for educators
v1.0 · ~20 pages

For classroom use. Single-class introduction (50-75 min), four-class unit, full-semester course (14 weeks). Lesson plans, discussion questions, classroom activities. Notes on adapting for different student populations.
Slogans & Quotes
v1.0 · ~15 pages

The strongest standalone lines from the manifesto. For posters, speeches, social media, presentations, tattoos. Three elevator pitches (five-line, ten-second, three-word).
The Funder's Brief
v1.0 · ~12 pages

For patient-capital sources — philanthropies, family foundations, public-sector grantmakers. The case, the foundation structure, three budget tiers (Seed $2-5M, Growth $15-30M, Foundational $60-120M), risks, action sequence.
The source
source.py
Python · the build source for the main manifesto

Edit this file to change the main manifesto. Regenerate index.html via the rebuild script shown in the file's comments. The main manifesto is the only document with a separate source file; the others are edited as HTML directly.

Suggested reading orders

If you want to be persuaded

essential.html → for_kids.html (for the emotional core) → index.html (Chapter VI stories) → slogans.html (the strongest lines)

If you want to argue back

essential.html → critics.html → index.html (whichever chapter the criticism led you to)

If you want to build something

essential.html → builder.html → index.html (Chapter X deployment scenarios)

If you want to teach this

for_kids.html → teaching.html → pamphlet.html → essential.html → index.html (each layer adds depth)

If you want to fund this

funders.html → essential.html → critics.html

If you want to extend the argument

new_ideas.html → critics.html → index.html (full)

If you want to share this widely

pamphlet.html (one page link), slogans.html (for social posts), essential.html (for serious readers)
